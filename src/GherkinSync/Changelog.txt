# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.2] - 2025-12-04

### Changed
- Modified the Pickle Index to allways assume NULL as value as in some scenarios the actual index value prevented proper scenario recognition in Azure DevOps.

## [1.3.1] - 2025-12-03

### Added

- This change log file
- Users can now add the Pickle Index to Associated Automation test names. This fixes detection issues in Azure DevOps for Scenario Outline tests using Reqnroll >= 3.0, where an extra method parameter prevented proper scenario recognition.
